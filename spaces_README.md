# Movie Shorts

Multimodal Video Understanding for Short-Form Content Engagement

Upload a video and automatically find the most engaging segments for YouTube Shorts, TikTok, and Instagram Reels.

## Features

- 🎯 **Multimodal Analysis**: Text, Audio, and Visual features
- 🔬 **Classical CV**: Edge detection, motion estimation, scene detection
- 📊 **Ablation Studies**: Compare different feature combinations
- ✂️ **Video Cutting**: Export engaging segments
